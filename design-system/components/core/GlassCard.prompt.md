The brand's signature surface: a blurred, saturated glass panel with a faint cyan-tinted border, used for any content raised off the Slate-950 background (cards, quote blocks, cover title plates).

```jsx
<GlassCard strong glow>
  <h3>Governance-Schicht für KI-Agenten</h3>
</GlassCard>
```

`strong` increases blur/saturation for a denser panel (e.g. cover title block). `glow` adds the ambient cyan box-shadow — use sparingly, one glowing element per view.
