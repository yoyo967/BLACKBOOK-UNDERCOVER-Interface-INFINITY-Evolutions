Pill-shaped button in three registers — solid cyan-fill primary (CTA), outlined secondary, and text-only ghost — for use on the dark Slate-950 surface.

```jsx
<Button variant="primary" size="md" onClick={() => {}}>Explore the repo</Button>
```

Variants: `primary` (cyan fill, dark text, glow), `secondary` (outlined, brightens border on hover), `ghost` (cyan text, no border). Sizes: `sm` / `md` / `lg`. Set `disabled` to dim to 45% opacity.
